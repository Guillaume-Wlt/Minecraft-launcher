## Requirements

- **Java 21 or higher** — [Download here](https://www.oracle.com/java/technologies/downloads/#java21)
- Windows 10/11

## Installation

1. Download `minecraft-launcher-1.0.jar`
2. Place it in a folder of your choice (e.g. `C:\MinecraftLauncher\`)
3. In the same folder, create a file named `start.bat` with the following content:

```bat
@echo off
java -jar minecraft-launcher-1.0.jar
pause
```

4. Double-click `start.bat` to launch the application

## Verify your Java version

Open a terminal and run:

```
java -version
```

You should see `java version "21"` or higher.
